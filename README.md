## 使用技術

- 後端 - Spring Boot( Spring MVC + Spring Security, Spring Data JPA)
- 資料庫 - MySQL
- 前端 - BootStrap, Thymeleaf

## 須自行設定項目
- Application.properties內mysql連線
- 建立db(src/main/resources/static/create_script.sql)
- 請以maven專案打開
